************ Lancement de l'application *************

1) backend
- Sous Eclipse:
	- Se positionner sur la classe principale TodoApplication.
	- Faire un clic droit et choisir run As : Spring boot app.
	- Le serveur back tournera sur le port 8080.
	

2) FrontEnd
- Faire un npm i pour installer les dépendences nécessaires au lancement du projet angular.
- Lancer la commande npm run start qui permettra de lancer le front en utilisant la configuration proxy qui pointe vers l'api backend.
- L'application tournera sur le port 4200.
- Consulter l'url localhost:4200 pour pouvoir visiter l'application.
